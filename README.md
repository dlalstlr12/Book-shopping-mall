## 프로젝트 소개
도서 구매 쇼핑몰을 만들기 위한 API를 설계하고 직접 구현한 프로젝트입니다.
Postman 도구를 활용하여 API의 정상 작동을 보장하며 API 엔드포인트를 구현하고, 사용자 인증, 도서 검색, 주문 처리 및 결제 시스템을 개발하였습니다.
또한, API에 대한 문서를 작성하여 쉽게 이해하고 사용할 수 있도록 하였습니다.
